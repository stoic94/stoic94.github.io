##Docker轻量级容器技术：

###docker的概念

docker 主机：安装了Docker中的主机
docker客户端：通过命令行访问host
docker仓库：用来保存打包好的镜像
docker镜像：客户机将仓库中的镜像下载到本地，运行就有了容器。
dcker容器：

###Docker的常用命令：
1.镜像操作：
docker search：搜索Docker仓库（docker hub）中所有的对象，tag表示镜像版本
docker pull :下载镜像 例如：docker pull ：5.5
docker images：查看本机所有镜像
docker rmi image-id：删除镜像
2.容器操作：
镜像运行后就产生了容器，从相当于安装软件后运行。

   >docker run --name mytomcat -d tomcat:latest：以mytomcat之名运行Tomcat container
    docker ps：查询正在运行的container
   docker ps -a：查询所有的container
   docker stop 容器id ：kill some container
   dokcer start 容器id： start some container
   docker rm 容器id: remove some container
   docker run -d -p 例子：  docker run -d -p 8888:8080 tomcat
-d：表示后台运行
-p 8888：8080 将主机的端口8888映射到容器的端口8080，所以访问主机的8888端口相当于访问容器的8080端口。
查看linux防火墙service fire status

docker logs container/container-id:查看docker的日志
